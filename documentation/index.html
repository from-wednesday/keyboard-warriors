Keyboard Warriors — Developer Reference

Shortcut Token Reference

Complete specification for the mac_sc and pc_sc fields in shortcut JSON files. Covers every detectable key, accepted synonyms, chip display behaviour, and OS differences.

Key: TOKEN canonical token SYNONYM accepted alias Mac Mac only PC PC only Both both platforms Extended chip pill, not graphical keyboard chip how it renders

01 — Fundamentals

Structure

Tokens are combined with + to form a shortcut string. Order does not matter — the parser normalises modifiers before matching.

// Single key "mac_sc": "CMD+S" // Multiple modifiers "mac_sc": "CMD+SHIFT+Z" // Pipe = accepted alternatives (first is shown on the chip) "mac_sc": "CMD+ARROWLEFT|HOME" // Key does not exist on this platform "pc_sc": "NULL"
Pipe alternatives. The system accepts any alternative listed after | as a correct answer. The chip always shows only the first alternative. Use this when a shortcut can be performed more than one way.
NULL. Use the literal value NULL (not an empty string) when a shortcut does not exist on a given platform. This prevents the question appearing on that OS.

02 — Modifier Keys

Modifiers

Modifiers are always rendered as a chip. They appear on the graphical keyboard. Left and right variants are interchangeable — holding either satisfies the requirement.

Token Synonyms Mac chip PC chip Platform Notes
CMD COMMAND CTRL ⌘ Ctrl Both Maps to ⌘ on Mac, Ctrl on PC. CTRL is accepted in JSON for readability.
SHIFT — ⇧ ⇧ Both
OPTION OPT ALT ⌥ Alt Both Use ALT freely in JSON; the chip adapts to the platform.
CONTROL — ⌃ — Mac The physical Control key on Mac, distinct from CMD. Not used on PC.
WIN WINDOWS — Win PC Windows key. Not used on Mac.
FN FUNCTION fn FN Both Only present on keyboards configured to show the Fn key.
CMD vs CONTROL on Mac. CMD maps to the ⌘ Command key. CONTROL maps to the ⌃ Control key — a separate physical key used in a small number of Mac shortcuts (e.g. CONTROL+CMD+V for Paste Special). Do not confuse them.

04 — Letters, Digits & Function Keys

Letters, Digits & Function Keys

All appear on the graphical keyboard. Use the character or label directly — no special wrapper token needed.

Letters

Write A through Z. Always uppercase in the token string. The chip shows the letter.

"CMD+S" → ⌘ S "CMD+Z" → ⌘ Z

Digits

Write 0 through 9. The chip shows the digit.

"CMD+1" → ⌘ 1 "CMD+0" → ⌘ 0

Function Keys — Graphical Keyboard (F1–F12)

F1 through F12 appear on the graphical keyboard and light up when pressed. Write the label directly.

F1 F2 F3 F4 F5 F6 F7 F8 F9 F10 F11 F12

Function Keys — Extended Chip Only (F13–F24) Extended

F13–F24 are not on the graphical keyboard. They appear as a chip pill when pressed. On Mac extended keyboards, the keys physically labelled PrintScreen/ScrollLock/Pause report as F13/F14/F15 to the browser.

F13 F14 F15 F16 F17 F18 F19 F20 F21 F22 F23 F24

Mac vs PC function keys. On Mac, F1–F12 are system media/brightness keys by default. The user may need to hold Fn to send the raw F-key signal to the browser. On PC, F-keys send directly. Use the CONFLICTS JSON files to mark keys that your OS or browser intercepts.

05 — Punctuation & Symbol Keys

Punctuation & Symbol Keys

These keys exist on the graphical keyboard. There are two categories: physical key tokens that refer to the key's unshifted character, and display override tokens that show the shifted character on the chip. Both match the same physical key regardless of whether Shift is held — the token only affects what the chip displays.

07 — Numpad Keys

Numpad Keys Extended

All numpad keys show as extended chip pills. The system automatically lights the equivalent main keyboard key where one exists (e.g. Numpad 5 lights the 5 key on the graphical keyboard).

No JSON changes needed for + shortcuts. The EQUAL / PLUS token already accepts both the main keyboard = key and the numpad Add key as correct answers. You only need NUMPADADD if you specifically want to require the numpad key.

09 — Quick Examples

Example Shortcut Definitions

// Standard cross-platform shortcut { "display": "save the document", "mac_sc": "CMD+S", "pc_sc": "CMD+S" } // Different shortcut per platform { "display": "open find and replace", "mac_sc": "CMD+SHIFT+H", "pc_sc": "CMD+H" } // Mac-only shortcut { "display": "quit the application", "mac_sc": "CMD+Q", "pc_sc": "NULL" } // Pipe: multiple accepted answers, first shown on chip { "display": "go to line start", "mac_sc": "CMD+ARROWLEFT|HOME", "pc_sc": "HOME" } // Display override: chip shows the shifted symbol { "display": "increase font size", "mac_sc": "CMD+SHIFT+MORETHAN", "pc_sc": "CMD+SHIFT+MORETHAN" } // Mac Control key (distinct from CMD) { "display": "paste special", "mac_sc": "CONTROL+CMD+V", "pc_sc": "CMD+ALT+V" } // F-key shortcut { "display": "open spell check", "mac_sc": "F7", "pc_sc": "F7" } // Extended key (chip pill only) { "display": "go to beginning of document", "mac_sc": "CMD+HOME", "pc_sc": "CMD+HOME" }
